Super heroes Test task
## Description
Small web server for creating super hero profile with photos
main tools:
nest js
PostgreSQL database
Sequalize for working with DB
S3 aws bucket for uploading photos
Swagger for Api docs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
